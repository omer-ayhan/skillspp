# Security Policy

## Supported Versions

Security fixes are applied to the latest state of the `main` branch.

## Reporting a Vulnerability

Please do not disclose vulnerabilities publicly in issues, pull requests, or
discussions.

Use GitHub Security Advisories for private reporting:

- https://github.com/omer-ayhan/skillspp/security/advisories/new

Include:

- A clear description of the issue.
- Impact and affected area.
- Reproduction steps or proof-of-concept.
- Suggested remediation if available.

You can expect an acknowledgement within 3 business days.
